# 投研分析模板库

## 模板 1: 创业信号分析

```markdown
## 分析维度
1. **融资动态** - 新一轮融资、估值变化、投资方
2. **产品发布** - 新产品、功能更新、技术突破
3. **团队变动** - 核心人员加入/离开、招聘信息
4. **战略合作** - 合作伙伴、行业联盟

## 评分标准 (1-5分)
- 5分: 明确的重大事件（融资、收购）
- 4分: 强信号（产品发布、核心招聘）
- 3分: 中等信号（行业观点、技术分享）
- 2分: 弱信号（日常动态）
- 1分: 无关内容

## 关键词
融资, Series, 估值, 收购, 上市, IPO, 发布, launch, 招聘, hiring, 合作, partnership
```

## 模板 2: 技术趋势追踪

```markdown
## 分析维度
1. **技术突破** - 论文、开源项目、性能提升
2. **应用落地** - 商业案例、行业应用
3. **行业观点** - 专家预测、趋势判断
4. **竞争态势** - 竞品动态、市场份额

## 评分标准 (1-5分)
- 5分: 革命性技术突破
- 4分: 重要技术进展
- 3分: 有价值的行业洞察
- 2分: 一般技术讨论
- 1分: 无关内容

## 关键词
breakthrough, SOTA, benchmark, open-source, demo, paper, research
```

## 模板 3: 投资人观点追踪

```markdown
## 分析维度
1. **投资逻辑** - 看好/看空的理由
2. **行业判断** - 对特定赛道的观点
3. **项目评价** - 对具体公司/产品的看法
4. **宏观洞察** - 市场趋势、政策影响

## 评分标准 (1-5分)
- 5分: 明确的投资决策信号
- 4分: 有深度的行业洞察
- 3分: 有参考价值的观点
- 2分: 一般性评论
- 1分: 无关内容

## 关键词
投资, invest, portfolio, thesis, bullish, bearish, 看好, 看空, 赛道
```

## 动态提示词生成规则

当用户输入模糊需求时，AnalysisGenerator 应按以下逻辑生成专属提示词：

1. **识别核心主题**: 从用户查询中提取关键领域（如"具身智能"→机器人、AI硬件）
2. **匹配基础模板**: 选择最接近的预设模板
3. **定制分析维度**: 根据主题调整分析角度
4. **生成关键词列表**: 结合行业知识扩展搜索词
5. **设定输出格式**: 根据用户习惯决定报告结构
